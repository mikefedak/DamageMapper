Instructions:

Viewing: Click on any of the thumbnails on the map to view a larger image as well as details

Adding: Use the red button in the bottom right corner to begin adding features. Once you click on the button, editing mode is triggered and the cursor changes to a pointer. Click anywhere on the map to add an 'event' and trigger the input form. You can press the 'esc' key any time to exit this mode The form allows two ways of uploading images:

1. Using the choose file button to upload from your device

2. Using a link to an existing image (note that this not work with images from Dominica News Online since they do not allow CORS). This however does work with twitter, instagram, and facebook images You can only upload one image at a time.

Once filled in, you can click save and if successful, the form will close. If it fails, you will get an error message.

Contact us in case of errors or if you have suggestions.
